# SCT_WD_1
A responsive single-page company website built using HTML, CSS, and JavaScript with modern UI design and smooth scroll-based interactions.
## Features

* Responsive single-page website layout
* Fixed navigation bar with smooth scroll effect
* Dynamic navbar style change on scroll using JavaScript
* Modern UI with gradient backgrounds and glassmorphism cards
* Hover animations for navigation links and cards
* Well-structured sections: Home, About, Services, and Contact
* Clean and readable code using pure HTML, CSS, and JavaScript
* Suitable for company landing page, portfolio, or internship project

## How to Run

1. Download or clone the repository

   ```bash
   git clone https://github.com/your-username/repository-name.git
   ```
2. Open the project folder
3. Double-click the `index.html` file
   **OR**
   Open `index.html` using any modern web browser (Chrome, Edge, Firefox)
4. The website will run locally in your browser without any additional setup
